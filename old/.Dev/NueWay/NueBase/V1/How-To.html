1. Installation
You can install NueBase by cloning the GitHub repo:
  1. GitHub Repository: https://github.com/NueWay/nuebase-nb--v1.0.1-stable
  2. Git Address: https://github.com/NueWay/nuebase-nb--v1.0.1-stable.git
After cloning, follow these steps:
  1. 1. Place the NueStuff directory in your system programs folder (e.g., C:\Program Files\NueStuff).
  2. 2. Ensure that the .\nb.exe file is accessible from your command line.
2. Features
With NueBase, you can easily create and run NueScript programs with the help of our trusty .\nb.exe sidekick. It's like having a personal assistant who's always ready to help you with your projects. Whether you're coding, running scripts, or managing configurations, NueBase has got your back. üöÄ
3. Usage
Here are some common commands you can use with NueBase:
  1. .\nb.exe new <projectname>: Create a new project
  2. .\nb.exe run: Run the default main file
  3. .\nb.exe run -s <scriptname>: Run a specific script
  4. .\nb.exe run --ignore-dev: Run without showing errors for missing dependencies
  5. .\nb.exe run -s <scriptname> --ignore-dev: Run a specific script ignoring missing dependencies
  6. .\nb.exe del <projectname>: Delete a project
  7. .\nb.exe conf init <projectname>: Initialize a new configuration file
  8. .\nb.exe conf <name> <value>: Update configuration
  9. .\nb.exe conf -s <scriptname> <scriptlocation>: Add a script to configuration
  10. .\nb.exe conf -s -cc: Check if all script files are present
  11. .\nb.exe conf dev: Update dependencies
  12. .\nb.exe conf dev -cc: Check if all libraries are present
4. Configuration
NueBase uses a nue-conf.json file for project configuration. You can:
  1. 1. Initialize a new configuration file
  2. 2. Add or update configuration values
  3. 3. Add scripts to the configuration
  4. 4. Check if all script files are present in their registered directories
5. Dependency Management
NueBase offers commands to manage your project dependencies:
  1. .\nb.exe conf dev: Update dependencies with files in the nuepkgs directory
  2. .\nb.exe conf dev -cc: Check if all libraries are present in their correct paths
  3. .\nb.exe run --ignore-dev: Run the program without showing errors for missing dependencies
6. Troubleshooting
If you encounter issues:
  1. 1. Ensure NueStuff is in the correct directory (e.g., C:\Program Files\NueStuff)
  2. 2. Check that your nue-conf.json file is properly configured
  3. 3. Use the --ignore-dev flag to run programs even with missing dependencies
  4. 4. Verify that all required .pkg files are present in the nuepkgs directory
7. Updates
Recent updates include:
  1. 1. Added conf dev and conf dev -cc commands for easier .pkg file management
  2. 2. Introduced run --ignore-dev and run <scriptname> --ignore-dev commands to run programs even with missing dependencies
