If you've ever thought, "Gee, C# is great, but it could use a bit more pizzazz and a sprinkle of modern magic," then C% is here to turn that daydream into reality.         int GCD(int a, int b) {
            while (b != 0) {
                int t = b;
                b = a % b;
                a = t;
            }
            return a;
        }
    
        public void Main() {
            System.Console.WriteLine(new Euclides().GCD(135, 63));
        }
