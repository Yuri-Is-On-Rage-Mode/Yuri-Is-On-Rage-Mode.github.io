Learn NueBase Commands!
nb new
Create New Project: Use nb new <projectname> to create a new project with the given name.

nb new MyNewProject

Requirements:
- Current Directory: The command can be run from any directory.
- Arguments: ProjectName
Possible Errors:
- "Project already exists" if a project with the same name already exists in the current directory.
nb run
Run Default Main File: Use nb run to execute the default main file specified in nue-conf.json.

nb run

Requirements:
- Current Directory: The command must be run from the project's root directory.
- Arguments: None
Possible Errors:
- "No configuration file found" if the nue-conf.json file is missing.
- "Main file not specified" if no default main file is set in the configuration.
nb run -s
Run Specific Script: Use nb run -s <scriptname> to run a specific script specified in nue-conf.json.

nb run -s MyScript

Requirements:
- Current Directory: The command must be run from the project's root directory.
- Arguments: ScriptName
Possible Errors:
- "Script not found" if the specified script is not in the configuration file.
nb run --ignore-dev
Run Without Dev Errors: Use nb run --ignore-dev to run the program without showing errors for missing dependencies.

nb run --ignore-dev

Requirements:
- Current Directory: The command must be run from the project's root directory.
- Arguments: None
Possible Errors:
- "No configuration file found" if the nue-conf.json file is missing.
- "Project file not found" if the main project file is missing.
nb run -s --ignore-dev
Run Specific Script Without Dev Errors: Use nb run -s <scriptname> --ignore-dev to run a specific script without showing errors for missing dependencies.

nb run -s MyScript --ignore-dev

Requirements:
- Current Directory: The command must be run from the project's root directory.
- Arguments: ScriptName
Possible Errors:
- "Script not found" if the specified script is not in the scripts/ directory.
nb del
Delete Project: Use nb del <projectname> to delete the project directory.

nb del MyProject

Requirements:
- Current Directory: The command can be run from any directory.
- Arguments: ProjectName
Possible Errors:
- "Project not found" if the specified project directory doesn't exist.
nb conf init
Initialize Configuration: Use nb conf init <projectname> to initialize a new nue-conf.json file for the project.

nb conf init MyNewProject

Requirements:
- Current Directory: The command should be run from the directory where you want to create the project.
- Arguments: ProjectName
Possible Errors:
- "Configuration file already exists" if nue-conf.json is already present in the directory.
nb conf
Update Configuration: Use nb conf <name> <value> to update nue-conf.json with the specified name and value.

nb conf mainFile app.nue

Requirements:
- Current Directory: The command must be run from the project's root directory.
- Arguments: Name Value
Possible Errors:
- "Configuration file not found" if nue-conf.json is missing.
- "Invalid configuration key" if the specified name is not a valid configuration option.
nb conf -s
Add Script to Configuration: Use nb conf -s <scriptname> <scriptlocation> to add a script to nue-conf.json.

nb conf -s start ./scripts/start.nue

Requirements:
- Current Directory: The command must be run from the project's root directory.
- Arguments: ScriptName ScriptLocation
Possible Errors:
- "Configuration file not found" if nue-conf.json is missing.
- "Script already exists" if a script with the same name is already in the configuration.
nb conf -s -cc
Check Script Files: Use nb conf -s -cc to check if all script files are present in their registered directories.

nb conf -s -cc

Requirements:
- Current Directory: The command must be run from the project's root directory.
- Arguments: None
Possible Errors:
- "Configuration file not found" if the nue-conf.json file is missing.
- "Script file missing" if any registered script is not found in its specified location.
nb conf dev
Update Dependencies: Use nb conf dev to update dependencies with files in the nuepkgs directory.

nb conf dev

Requirements:
- Current Directory: The command must be run from the project's root directory.
- Arguments: None
Possible Errors:
- "nuepkgs directory not found" if the nuepkgs directory is missing.
- "Failed to update dependencies" if there's an issue updating the dependencies.
nb conf dev -cc
Check Libraries: Use nb conf dev -cc to check if all libraries are present in their correct paths.

nb conf dev -cc

Requirements:
- Current Directory: The command must be run from the project's root directory.
- Arguments: None
Possible Errors:
- "Configuration file not found" if the nue-conf.json file is missing.
- "Library file missing" if any required library is not found in its specified location.
