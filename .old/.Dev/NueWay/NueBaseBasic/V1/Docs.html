Learn NBB Commands!
nbb new
Create New Project: Use nbb new <projectname> to create a new project with the given name.

nbb new MyNewProject

Requirements:
- Current Directory: The command can be run from any directory.
- Arguments: ProjectName
Possible Errors:
- "Project already exists" if a project with the same name already exists in the current directory.
nbb run
Run Default Main File: Use nbb run to execute the default main file specified in nue-conf.nueon.

nbb run

Requirements:
- Current Directory: The command must be run from the project's root directory.
- Arguments: None
Possible Errors:
- "No configuration file found" if the nue-conf.nueon file is missing.
- "Main file not specified" if no default main file is set in the configuration.
nbb run -s
Run Specific Script: Use nbb run -s <scriptname> to run a specific script specified in nue-conf.nueon.

nbb run -s MyScript

Requirements:
- Current Directory: The command must be run from the project's root directory.
- Arguments: ScriptName
Possible Errors:
- "Script not found" if the specified script is not in the configuration file.
nbb run --ignore-dev
Run Without Dev Errors: Use nbb run --ignore-dev to run the program without showing errors for missing dependencies.

nbb run --ignore-dev

Requirements:
- Current Directory: The command must be run from the project's root directory.
- Arguments: None
Possible Errors:
- "No configuration file found" if the nue-conf.nueon file is missing.
- "Project file not found" if the main project file is missing.
nbb run -s --ignore-dev
Run Specific Script Without Dev Errors: Use nbb run -s <scriptname> --ignore-dev to run a specific script without showing errors for missing dependencies.

nbb run -s MyScript --ignore-dev

Requirements:
- Current Directory: The command must be run from the project's root directory.
- Arguments: ScriptName
Possible Errors:
- "Script not found" if the specified script is not in the scripts/ directory.
nbb del
Delete Project: Use nbb del <projectname> to delete the project directory.

nbb del MyProject

Requirements:
- Current Directory: The command can be run from any directory.
- Arguments: ProjectName
Possible Errors:
- "Project not found" if the specified project directory doesn't exist.
nbb conf init
Initialize Configuration: Use nbb conf init <projectname> to initialize a new nue-conf.nueon file for the project.

nbb conf init MyNewProject

Requirements:
- Current Directory: The command should be run from the directory where you want to create the project.
- Arguments: ProjectName
Possible Errors:
- "Configuration file already exists" if nue-conf.nueon is already present in the directory.
nbb conf
Update Configuration: Use nbb conf <name> <value> to update nue-conf.nueon with the specified name and value.

nbb conf mainFile app.nue

Requirements:
- Current Directory: The command must be run from the project's root directory.
- Arguments: Name Value
Possible Errors:
- "Configuration file not found" if nue-conf.nueon is missing.
- "Invalid configuration key" if the specified name is not a valid configuration option.
nbb conf -s
Add Script to Configuration: Use nbb conf -s <scriptname> <scriptlocation> to add a script to nue-conf.nueon.

nbb conf -s start ./scripts/start.nue

Requirements:
- Current Directory: The command must be run from the project's root directory.
- Arguments: ScriptName ScriptLocation
Possible Errors:
- "Configuration file not found" if nue-conf.nueon is missing.
- "Script already exists" if a script with the same name is already in the configuration.
nbb conf -s -cc
Check Script Files: Use nbb conf -s -cc to check if all script files are present in their registered directories.

nbb conf -s -cc

Requirements:
- Current Directory: The command must be run from the project's root directory.
- Arguments: None
Possible Errors:
- "Configuration file not found" if the nue-conf.nueon file is missing.
- "Script file missing" if any registered script is not found in its specified location.
nbb conf dev
Update Dependencies: Use nbb conf dev to update dependencies with files in the nuepkgs directory.

nbb conf dev

Requirements:
- Current Directory: The command must be run from the project's root directory.
- Arguments: None
Possible Errors:
- "nuepkgs directory not found" if the nuepkgs directory is missing.
- "Failed to update dependencies" if there's an issue updating the dependencies.
nbb conf dev -cc
Check Libraries: Use nbb conf dev -cc to check if all libraries are present in their correct paths.

nbb conf dev -cc

Requirements:
- Current Directory: The command must be run from the project's root directory.
- Arguments: None
Possible Errors:
- "Configuration file not found" if the nue-conf.nueon file is missing.
- "Library file missing" if any required library is not found in its specified location.
